# Mailer
## _Node, Nest, SendGrid_
![N|Solid](https://i.ytimg.com/vi/_LE4UwBEOP4/maxresdefault.jpg)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
